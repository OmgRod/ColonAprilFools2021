# 170th Secret Coin (April Fools)

![Mod logo](/logo.png)

This mod "adds" the 170th secret coin into the game.

For help with the mod, I recommend you watch the following YouTube videos:

- [April Fools Video](https://www.youtube.com/watch?v=h7UD0lotTVM)
- [Behind the Scenes](https://www.youtube.com/watch?v=q9Bu6BZbuQ8)

## Setup

To build this mod, you need Geode SDK and a C++ compiler. To setup Geode, read the guide [here](https://gdresources.omgrod.me/geode/guides/setup/).

## Mod Info

### FAQ

1. Is the coin legit?

> Nope. The coin does exist in this mod, but clicking it won't increase your actual secret coin count.

2. How do I use this mod?

> Watch the video(s) above - IT'S AN APRIL FOOLS VIDEO SO DO NOT TAKE EVERYTHING COLON SAYS AS TRUE! (ignoring the parts mentioning Boomlings/email), or read instructions below!

### How to use this mod?

1. Go to the Vault of Secrets and type the code `destroy`.
2. Open the basement. If you freed the monster (or have enabled the mod setting to spoof the monster being freed), you will see Scratch.
3. Scratch will tell you to find RobTop and give him his emblem. Go to the garage (icon kit area) and click the lock.
4. There, the shopkeeper will speak to you. If you click the lock again, you will get a popup asking to enter a 15 character code (you do not need to buy a real Steam gift card for this section).
5. Once you entered the code, click the lock again to open RobTop's lair. Go through all the dialog with RobTop.
6. Go to the Vault and enter the code `CLUBSTEP` (in all caps).
7. Re-open the game. You will see some mysterious text. Then, the game will close.
8. Re-open the game and go to the Basement. There, you will see a coin. Click it (if you want)!
9. Exit the menu and read the very legitimate and totally up-to-date credits for the game (because of course Viprin is still part of the community).
